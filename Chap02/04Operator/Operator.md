## 算術演算子
| 演算子 | 説明     | 例     |
| ------ | -------- | ------ |
| +      | 加算     | 3 + 5  |
| -      | 減算     | 10 - 7 |
| *      | 乗算     | 2 * 4  |
| /      | 除算     | 8 / 4  |
| %      | 剰余     | 9 % 2  |
| ++     | 前置加算 | ++x;   |
| --     | 前置減算 | --x;   |
| ++     | 後置加算 | x++;   |
| --     | 後置減算 | x--;   |

## 代入演算子
| 演算子 | 説明                                               | 例     |
| ------ | -------------------------------------------------- | ------ |
| =      | 変数などに値を代入                                 | x = 1  |
| +=     | 左辺の値に右辺の値を加算したものを代入             | x+=1   |
| -=     | 左辺の値から右辺の値を減算したものを代入           | x-=2   |
| *=     | 左辺の値から右辺の値を乗算したものを代入           | x\*=3  |
| /=     | 左辺の値を右辺の値で乗算したものを代入             | x/=2   |
| %=     | 左辺の値を右辺の値で乗算した余りを代入             | x%=2   |
| &=     | 左辺の値を右辺の値で論理積演算した結果を代入       | x&=5   |
| \|=    | 左辺の値を右辺の値で論理和演算した結果を代入       | x\|=5  |
| ^=     | 左辺の値を右辺の値で排他的論理和演算した結果を代入 | x^=5   |
| <<=    | 左辺の値を右辺の値だけ左シフトした結果を代入       | x<<=1  |
| >>=    | 左辺の値を右辺の値だけ右シフトした結果を代入       | x>>=1  |
| >>>=   | 左辺の値を右辺の値だけ右シフトした結果を代入       | x>>>=1 |

## 比較演算子
| 演算子 | 説明                                                                    | 例            |
| ------ | ----------------------------------------------------------------------- | ------------- |
| ==     | 左辺と右辺が等しい場合はtrue                                            | 6 == 6        |
| !=     | 左辺と右辺が等しくない場合はtrue                                        | 5 != 5        |
| <      | 左辺が右辺より小さい場合はtrue                                          | 5 < 8         |
| >      | 左辺が右辺より大きい場合はtrue                                          | 8 > 9         |
| <=     | 左辺が右辺以下の場合はtrue                                              | 8 <= 8        |
| >=     | 左辺が右辺以上の場合はtrue                                              | 3 >= 9        |
| ===    | 左辺と右辺で値が等しく型も同じ場合はtrue                                | 5 === 5       |
| !==    | 左辺と右辺が等しくない場合、またはデータ型が異なる場合はtrue            | 6 !== 6.0     |
| ?:     | 「条件式? 式1: 式2」。条件式がtrueの場合は式1を、falseの場合は式2を返す | (x===1) ? 1:0 |


## 論理演算子
| 演算子 | 説明                                 | 例                             |
| ------ | ------------------------------------ | ------------------------------ |
| &&     | 左右の式がともにtrueの場合はtrue     | 100 === 100 && 1000 === 1000   |
| \|\    | 左右の式のどちらかがtrueの場合はtrue | 100 === 100 \|\| 1000 === 1000 |
| !      | 式がfalseの場合はtrue                | !(10 > 100)                    |

#### falsyな値
* 空文字
* 数値の0
* NaN
* null
* undefined

上記以外はすべてtrueとみなされる

#### ショートカット演算
論理積／論理和演算子を使用するとき、「左辺だけ評価されて、右辺が評価されない」caseがある点に注意する。「&&」は左辺がfalseと評価されたら右辺の評価は実行されない。このような演算をショートカット演算（短絡演算）という。

## ビット演算
整数値を2進数で表した場合の各桁に対して、論理計算をすること。

| 演算子 | 説明                                                                         | 例                                    |
| ------ | ---------------------------------------------------------------------------- | ------------------------------------- |
| &      | 左式と右式の両方にセットされているビット                                     | 10 & 5 => 1010 & 0101 => 0000 => 0    |
| \|     | 左式と右式のどちらかにセットされているビット                                 | 10 \| 5 => 1010 \| 0101 => 1111 => 15 |
| ^      | 左式と右式のどちらかにセットされていて、かつ、両方にセットされていないビット | 10 ^ 5 => 1010 ^ 0101 => 1111 => 15   |
| ~      | ビットを反転                                                                 | ~10 => ~1010 => 0101 => -11           |
| <<     | ビットを左にシフト                                                           | 10 << 1 => 1010 << 1 => 10100 => 20   |
| >>     | ビットを右にシフト                                                           | 10 >> 1 => 1010 >> 1 => 0101 => 5     |
| >>>    | ビットを右にシフトして左端を0で埋める                                        | 10 >>> 2 => 1010 >>> 2 => 0010 => 2   |
