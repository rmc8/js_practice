## 算術演算子
| 演算子 | 説明     | 例     |
| ------ | -------- | ------ |
| +      | 加算     | 3 + 5  |
| -      | 減算     | 10 - 7 |
| *      | 乗算     | 2 * 4  |
| /      | 除算     | 8 / 4  |
| %      | 剰余     | 9 % 2  |
| ++     | 前置加算 | ++x;   |
| --     | 前置減算 | --x;   |
| ++     | 後置加算 | x++;   |
| --     | 後置減算 | x--;   |
