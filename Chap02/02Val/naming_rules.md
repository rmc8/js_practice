## Naming Rules

| Rule                                                     | Good             | NG          |
| -------------------------------------------------------- | ---------------- | ----------- |
| 1文字目は\[a-zA-Z_$]であること                           | _name, $msg      | 1x          |
| 2文字目は1文字目で使える文字か、数字のいずれかであること | msg1, _name0     | c@c, name-0 |
| 変数に含まれる英字の大文字・小文字は区別される           | name, Name       | -           |
| JavaScriptの予約語でないこと                             | tiff, forth, def | for, if     |
